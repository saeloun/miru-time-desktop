# Miru Time Tracking 0.1.2

Miru Time Tracking 0.1.2 moves the desktop app fully under the Saeloun GitHub organization, switches local development and release work to Bun through mise, and refreshes dependencies for the next public desktop release.

## Downloads

- macOS arm64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.2/Miru.Time.Tracking-darwin-arm64-0.1.2.zip
- Linux x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.2/Miru.Time.Tracking-linux-x64-0.1.2.zip
- Windows x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.2/Miru.Time.Tracking-win32-x64-0.1.2.zip

## Highlights

- Canonical repository and release links now point to `saeloun/miru-time-desktop`.
- The repo now uses Bun through mise, with `bun.lock` as the package lockfile.
- Dependencies were updated to current compatible releases.
- A repo-local `AGENTS.md` documents desktop release, verification, product, and gbrain review expectations.
- The Electron auto-update source now points to the Saeloun repository.
- Electron E2E coverage now includes the signed-in idle recovery UI, RTL locale layout, and live timesheet history date switching.
- Bun audit is clean after top-level overrides for Electron Forge transitive packages.
