# Miru Time Tracking 0.1.4

Miru Time Tracking 0.1.4 is a rebuild and install verification release for the current desktop timer. It packages the latest timer stack, detailed entry drill-down, and Miru web sync behavior into fresh macOS, Linux, and Windows ZIP assets.

## Downloads

- macOS arm64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.4/Miru.Time.Tracking-darwin-arm64-0.1.4.zip
- Linux x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.4/Miru.Time.Tracking-linux-x64-0.1.4.zip
- Windows x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.4/Miru.Time.Tracking-win32-x64-0.1.4.zip

- Fixed the main timer hero layout so the elapsed time and icon controls do not overlap in the compact popup.
- Rebuilt the app from the current verified desktop timer code.
