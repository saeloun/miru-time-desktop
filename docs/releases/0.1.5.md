# Miru Time Tracking 0.1.5

Miru Time Tracking 0.1.5 is a UX and release refresh for the desktop timer. It tightens the main timer layout, adds a packaged Electron regression spec for the compact popup, refreshes screenshots, and ships Apple Silicon, Intel macOS, Linux, and Windows ZIP assets.

## Downloads

- macOS Apple Silicon ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.5/Miru.Time.Tracking-darwin-arm64-0.1.5.zip
- macOS Intel ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.5/Miru.Time.Tracking-darwin-x64-0.1.5.zip
- Linux x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.5/Miru.Time.Tracking-linux-x64-0.1.5.zip
- Windows x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.5/Miru.Time.Tracking-win32-x64-0.1.5.zip

## Highlights

- Improved the main timer hero with a fixed control dock and tighter elapsed-time typography.
- Added a packaged Electron layout spec that checks the elapsed timer and controls do not overlap.
- Kept the timer stack, detailed entry drill-down, idle recovery, and Miru web sync coverage intact.
- Refreshed README screenshots from the packaged desktop app surfaces.
- Built Apple Silicon, Intel macOS, Linux, and Windows ZIPs.
