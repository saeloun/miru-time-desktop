# Miru Time Tracking 0.1.6

Miru Time Tracking 0.1.6 fixes production login resilience for desktop users whose account login succeeds but a follow-up workspace refresh is temporarily unavailable.

- macOS Apple Silicon ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.6/Miru.Time.Tracking-darwin-arm64-0.1.6.zip
- macOS Intel ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.6/Miru.Time.Tracking-darwin-x64-0.1.6.zip
- Linux x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.6/Miru.Time.Tracking-linux-x64-0.1.6.zip
- Windows x64 ZIP: https://github.com/saeloun/miru-time-desktop/releases/download/v0.1.6/Miru.Time.Tracking-win32-x64-0.1.6.zip

## Fixes

- Keeps the desktop session signed in after a successful Miru password login even when `/api/v1/workspaces` is temporarily unavailable.
- Preserves profile refresh for avatar, locale, company, and role data without making that refresh invalidate the local-first timer session.
- Shows the account surface after login so the user can keep tracking locally while sync recovers.
